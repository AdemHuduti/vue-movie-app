# Movie application with Vue.js and Vuex

> A Vue.js project

## Quick Start

``` bash
# git clone
git clone: https://github.com/AdemHuduti/vue-movie-app.git

# cd vue-movie-app
cd vue-movie-app

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

# production build
npm run build
```
